# Meus Cronogramas de Internato

Aplicativo web instalável (PWA) com os cronogramas de Obstetrícia e Pediatria.

## Publicar no GitHub Pages

1. Crie um repositório no GitHub, por exemplo `cronogramas-internato`.
2. Envie todos os arquivos e a pasta `icons` para a raiz do repositório.
3. Abra **Settings > Pages**.
4. Em **Build and deployment**, escolha **Deploy from a branch**.
5. Selecione a branch **main**, a pasta **/(root)** e clique em **Save**.
6. Aguarde o GitHub mostrar o endereço publicado.

## Instalar no iPhone

1. Abra o endereço publicado no **Safari**.
2. Toque no botão **Compartilhar**.
3. Role as opções e toque em **Adicionar à Tela de Início**.
4. Confirme em **Adicionar**.

O aplicativo guardará a última especialidade, rodízio e grupo selecionados. Após a primeira abertura completa, o conteúdo principal também poderá ser consultado sem internet.

## Privacidade

O cronograma contém nomes de alunos e professores. Confirme se essas informações podem ficar acessíveis publicamente antes de ativar o GitHub Pages.
